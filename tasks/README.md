# tasks

## Usage

```bash
$ tasks --help
tasks 0.6.0

USAGE:
    tasks <SUBCOMMAND>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    add         Adds a new task
    complete    Completes the task
    help        Prints this message or the help of the given subcommand(s)
    list        Lists tasks
    remove      Removes the task
```
