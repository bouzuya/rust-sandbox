# week

## Installation

```console
$ version=0.2.1
$ curl -L "https://github.com/bouzuya/rust-sandbox/releases/download/week%2F${version}/week-x86_64-apple-darwin" > week
$ chmod +x week
$ ./week --help
week 0.2.1
Prints the week that contains the date

USAGE:
    week <DATE>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <DATE>    The date
```
